# ScoreCardCustomer
Project Bootscamp dibimbing DS 17 dengan topik Scorecard Customer untuk top up limit creditcard, data diperoleh dari kaggle dengan skillset: 
- Logistic Regression
- Gradient Boosting
- Weigh of Evidence (WOE)
- Information Value (IV)
- Binning
- Chi-Square Binning
